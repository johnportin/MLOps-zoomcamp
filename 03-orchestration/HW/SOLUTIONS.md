## 1: Run Mage
- First we run:
    ```
        git clone https://github.com/mage-ai/compose-quickstart.git mage-quickstart \   
        && cd mage-quickstart \
        && cp dev.env .env && rm dev.env \
        && docker compose up
    ```
- Then we can see `v0.9.71` in the top corner.
